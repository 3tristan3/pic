# Image Processing Workbench (图片处理工作台)

这是一个基于浏览器的纯前端图片处理工具，专为满足日常办公中的基础图片编辑需求而设计。

本项目无需后端服务器支持，所有图片处理逻辑（压缩、格式转换、裁剪等）均在用户浏览器本地完成，确保数据安全与隐私。

## 核心功能

本项目包含四个主要功能模块：

1.  **图片压缩 (Image Compression)**
    *   **智能调整**：根据设定的目标大小（KB），自动调整图片质量与尺寸。
    *   **批量处理**：支持通过弹窗管理图片库，一次性处理多张图片。
    *   **自动打包**：多张图片处理完成后，自动生成 ZIP 压缩包下载。

2.  **格式转换 (Format Conversion)**
    *   支持 JPG、PNG、WebP 三种常见格式的互转。
    *   在转换过程中尽可能保留原图分辨率与画质。

3.  **图片放大 (Image Upscaling)**
    *   基于 Canvas 的双三次插值算法（Bicubic Interpolation）。
    *   提供 1.5x、2.0x、3.0x 放大倍率，用于基础的图片尺寸放大（非 AI 超分）。

4.  **图片裁剪 (Interactive Cropping)**
    *   集成 Cropper.js，提供类似手机修图的交互体验。
    *   支持自由比例裁剪，或锁定 1:1、16:9、4:3 等常用比例。
    *   支持导出透明背景的 PNG 图片。
*   **纯前端实现**：无后端数据库，无 API 调用，图片不上传至云端。
